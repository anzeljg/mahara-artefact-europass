Lesing
  • A1: Jeg kan forstå vanlige navn, ord og veldig enkle setninger, f.eks. i annonser, på oppslag og plakater eller i brosjyrer.
  • A2: Jeg kan lese svært korte og enkle tekster. Jeg kan finne bestemte opplysninger jeg leter etter i vanlige tekster, som reklame, brosjyrer, menyer og rutetabeller, og jeg kan forstå korte, enkle personlige brev.
  • B1: Jeg kan forstå tekster som er skrevet i et dagligdags språk eller som forholder seg til jobben min. Jeg kan forstå innholdet i personlige brev hvor avsenderen beskriver sine opplevelser, følelser og ønsker.
  • B2: Jeg kan lese artikler og rapporter om aktuelle emner hvor forfatteren inntar en bestemt holdning til et problem eller uttrykker et bestemt synspunkt. Jeg kan forstå de fleste moderne noveller og lett tilgengelige romaner.
  • C1: Jeg kan forstå lang og innviklet faktatekst og skjønnlitterære tekster, og oppfatte forskjeller i stil. Jeg kan forstå fagspråk i artikler og tekniske instruksjoner, også når det ligger utenfor mitt felt.
  • C2: Jeg kan uten problemer lese nesten alle former for tekst, også abstrakte tekster som inneholder vanskelige ord og setningsstrukturer, f. eks. instruksjoner, fagartikler og litterære verk.
